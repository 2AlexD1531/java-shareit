Микросервисное приложение для совместного использования вещей. Пользователи могут сдавать вещи в аренду, бронировать предметы и оставлять отзывы.

**📦 Архитектура:**
| Компонент | Описание |
|-----------|----------|
| **Gateway** | API-шлюз для валидации и маршрутизации запросов |
| **Server** | Основной сервис с бизнес-логикой |
| **Database** | PostgreSQL для хранения данных |

**✨ Функционал:**
- 👤 **Пользователи** — регистрация, управление профилями
- 📦 **Вещи** — добавление, редактирование, поиск по названию/описанию
- 📅 **Бронирования** — аренда на конкретные даты, подтверждение/отклонение владельцем
- 📝 **Комментарии** — отзывы после завершения аренды
- 📨 **Запросы** — создание запросов на отсутствующие вещи
- 📄 **Пагинация** — постраничный вывод списков

**🛠️ Стек:** Java 21, Spring Boot 3, Hibernate/JPA, PostgreSQL, H2 (тесты), Docker, JUnit 5, Mockito, Maven
